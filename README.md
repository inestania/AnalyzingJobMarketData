# Analyzing Job Market Data
Tugas 3 Business Intelligence

Data
- Sumber data berasal dari studi kasus Analyzing Job Market Data.
- Data awal dalam format CSV sebelum dimuat ke Power BI.
- Dilakukan proses cleaning & transformation
- Data kemudian dimodelkan dalam Power BI untuk mendukung pembuatan visual dan perhitungan DAX.

# Navigasi Dashboard
- Home
- Jobs
- Skills
- Company

# Fitur Utama Dashboard
- Job Dashboard
Menampilkan ringkasan lowongan, tren posisi, dan distribusi job berdasarkan kategori, lokasi, maupun tingkat pengalaman.
- Skills Dashboard
Mengevaluasi keterampilan paling populer untuk berbagai peran, termasuk frekuensi kemunculan skill dalam tiap lowongan.
- Company Dashboard
Menampilkan perusahaan paling aktif merekrut, jenis peran yang dibuka, serta pola rekrutmen yang muncul dari dataset.

# Manfaat
- Memberikan wawasan mengenai keterampilan paling dibutuhkan di dunia kerja saat ini.
- Menjadi referensi bagi job seeker untuk menentukan skill yang perlu dikuasai.
- Memberikan pemahaman yang lebih jelas mengenai kebutuhan tenaga kerja berdasarkan tingkat pengalaman.
